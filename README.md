# 唱吧前端技术分享会

唱吧前端分享会是唱吧前端团队发起的涵盖前端、后端和其他知识领域的分享会，分享会创办自 2018 年 10 月，在创办伊始，我们致力于积极在公司内部分享知识，而今，我们希望能为更多的朋友带来有价值的内容，把知识传播的更远！

感谢大家的支持与厚爱。
