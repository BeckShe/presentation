Teleport

请选取图片区域

2021

唱吧前端团队分享会

  <teleport to='#modal'>
    <div id="center" v-if="dialogVisible">
      <h2>this is a modal dialog!</h2>
      <button @click="buttonClick">Close</button>
    </div>
  </teleport>

Vue 鼓励我们通过将 UI 和相关行为封装到组件中来构建 UI。我们可以将它们嵌套在另一个内部，以构建一个组成应用程序 UI 的树。 然而，有时组件模板的一部分逻辑上属于该组件，而从技术角度来看，最好将模板的这一部分移动到 DOM 中 Vue app 之外的其他位置。
